# Sample Documentation Repository

This is a sample repository containing markdown documentation files for testing the Uplifted Mascot ingestion pipeline.

## Contents

- **Getting Started Guide** - Introduction and quick start
- **API Reference** - Complete API documentation
- **Configuration Guide** - How to configure the system
- **Architecture Overview** - System design and components
- **Troubleshooting** - Common issues and solutions

This repository serves as a test dataset for:
- Document processing and chunking

- Vector search indexing

Use this to verify the ingestion pipeline works correctly before processing your actual documentation.
